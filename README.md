# python_assistant

This is a simple virtual assistant made using python.
Note : It is not a model trained to recognise wake words or such, its a simple assistant that uses Speech Recognition module and algorithms to take commands.

# Current Key Features

* Open Websites
* Open Files on computer
* Basic Google search
* Play music on Youtube
* Date and Time support
